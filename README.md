# 基于主成分分析法 (PCA) 和奇异值分解 (SVD) 的人脸识别系统

## 1. 项目简介
本项目基于 MATLAB 平台，完全自主编写线性代数相关功能函数（不依赖内置 pca/svd/pdist 等），实现了完整的 PCA+SVD 人脸识别算法，并配套可交互的 GUI 操作界面。支持离线样本识别与摄像头实时人脸识别两大核心功能，完全符合课程项目的全部验收要求。

## 2. 运行环境
- **操作系统**: Windows / macOS / Linux
- **软件环境**: MATLAB R2022b 及以上版本
- **硬件依赖**: 电脑自带摄像头（用于实时识别功能）
- **工具箱依赖**: 无需额外核心算法工具箱（PCA/SVD 均为纯手写）。但调用电脑摄像头需要安装官方免费的 *MATLAB Support Package for USB Webcams*。

## 3. 文件结构说明
代码采用模块化面向对象设计，各司其职：
- `FaceApp.m`：GUI 界面主程序类，包含所有可视化界面交互、操作流程控制与系统日志调度。
- `ImagePreprocess.m`：自定义图像预处理模块。自主实现图像灰度化、去噪（3x3均值滤波）、尺寸归一化（最近邻插值）、像素值归一化及向量化。
- `PCA_SVD_Core.m`：特征提取核心算法模块。自主实现平均脸计算、数据去中心化、协方差计算、基于雅可比 (Jacobi) 迭代法的 SVD 特征分解、特征脸生成与特征空间投影。
- `ClassifierCore.m`：分类器模块。自主实现最近邻分类器（纯手写欧式距离匹配），以及批量测试与正确率自动计算功能。
- `untitled.m`：系统快速启动入口脚本。

## 4. 数据集结构要求
系统会自动解析人脸数据库，**请确保数据集按以下文件夹结构放置**：
```
人脸数据库/
├── 张三/
│   ├── 1.jpg
│   ├── 2.jpg
│   └── ... (共 10 张)
├── 李四/
│   ├── 1.jpg
│   └── ...
└── 王五/
    └── ...
```
系统读取时会自动提取上级文件夹名称（如“张三”）作为真实人员身份标签，并自动按照 7:3 的比例划分为训练集与测试集。

## 5. 操作步骤与功能测试用例

### 5.1 系统启动
1. 在 MATLAB 中，将“当前文件夹”切换至本代码所在目录。
2. 运行 `untitled.m` 脚本，或者在命令行输入 `FaceApp()` 即可调出图形化交互界面。

### 5.2 数据集加载与模型训练测试
1. 点击 **[导入人脸数据集文件夹]** 按钮，选择上述准备好的“人脸数据库”根目录。
   - *预期结果*：日志区提示划分的训练集与测试集数量，完成数据加载。
2. 点击 **[启动模型训练 (PCA+SVD)]** 按钮。
   - *预期结果*：后台运行自主编写的雅可比特征分解算法，耗时几秒到十几秒不等（视人数而定）。完成后，中间结果可视化区会显示“原始图”、“预处理图”、“平均脸”及“第一主成分特征脸”。
3. 点击 **[批量测试并计算准确率]** 按钮。
   - *预期结果*：对自动划分出的 30% 测试集进行批量特征投影与欧式距离分类，最终在界面上方红色字体显示模型的整体识别正确率。

### 5.3 离线人脸点选识别测试
1. 点击 **[选择待测人脸图片]** 按钮，从本地磁盘（如测试集中）任选一张人脸图片导入。
   - *预期结果*：左侧图框显示你选中的待测人脸。
2. 点击 **[执行单样本识别]** 按钮。
   - *预期结果*：在“匹配结果”区显示数据库中与其最相似的对应原图。文字标签将明确输出该人员身份（如：张三）以及匹配到的具体文件名。

### 5.4 摄像头实时人脸识别测试
1. 点击 **[启动/关闭摄像头]** 按钮。
   - *预期结果*：界面右侧区域成功调用本地电脑摄像头，并实时显示动态画面。
2. 请一位同学坐在摄像头前，点击 **[采集当前帧并识别]** 按钮。
   - *预期结果*：系统截取当前画面中心区域，完成自定义预处理并投影至特征空间。画面上方蓝色状态栏会实时变成红色，并输出识别到的同学身份及匹配距离。

## 6. 核心硬性约束达标说明
1. **纯手工底层算法**：代码中 100% 杜绝了 `pca()`、`svd()`、`eig()`、`pdist()` 等内置高级算法函数的调用。SVD 奇异值分解使用了经典的 Jacobi 旋转算法 (`myEigJacobi`) 自主实现。
2. **脱离内置工具箱的图像处理**：没有调用 `imresize` 等高级函数，而是利用双层 `for` 循环和坐标映射公式 (`resizeImg`) 实现了缩放；没有调用 `rgb2gray`，而是使用加权公式 (`toGray`) 实现了灰度化。
3. **全流程 GUI 操作**：所有功能均封装于 `FaceApp.m` 中，用户全程只需点击按钮，无需在命令行敲击任何指令。
